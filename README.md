# Heart-Disease-Prediction-EDA
This project performs Exploratory Data Analysis (EDA) on a Heart Disease dataset using Python.
The goal is to understand patterns, relationships, and important features that may influence heart disease prediction.

The analysis includes:

Data inspection
Statistical summaries
Missing value checks
Correlation analysis
Distribution visualizations
